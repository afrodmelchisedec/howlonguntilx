
=============================================================
STRIPE SETUP GUIDE
=============================================================

1. Create account at https://stripe.com
2. Go to Developers → API keys
3. Copy Secret key and add to .env:

STRIPE_SECRET_KEY="sk_test_..."

4. Create a Product:
   - Go to Products → Add product
   - Name: HowLongUntil Premium
   - Price: $4.00 / month (recurring)
   - Copy the Price ID (price_xxxxx)

STRIPE_PRO_MONTHLY_PRICE_ID="price_..."

5. Set up webhook (for local testing, use Stripe CLI):
   stripe listen --forward-to localhost:3001/api/stripe/webhook
   Copy the webhook signing secret:

STRIPE_WEBHOOK_SECRET="whsec_..."

6. For production webhook:
   - Go to Developers → Webhooks → Add endpoint
   - URL: https://yourdomain.com/api/stripe/webhook
   - Events: checkout.session.completed, customer.subscription.deleted

=============================================================
FULL .env additions needed:
=============================================================

STRIPE_SECRET_KEY=""
STRIPE_PRO_MONTHLY_PRICE_ID=""
STRIPE_WEBHOOK_SECRET=""
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=""
