=============================================================
HOW TO SET UP GOOGLE OAUTH
=============================================================

1. Go to: https://console.cloud.google.com/
2. Create a new project (or select existing)
3. Go to APIs & Services → Credentials
4. Click "Create Credentials" → "OAuth 2.0 Client IDs"
5. Application type: Web application
6. Add Authorized redirect URIs:
   http://localhost:3001/api/auth/callback/google
   http://localhost:3000/api/auth/callback/google
   https://yourdomain.com/api/auth/callback/google  (when live)
7. Copy Client ID and Client Secret
8. Add to your .env file:

GOOGLE_CLIENT_ID="your-client-id-here.apps.googleusercontent.com"
GOOGLE_CLIENT_SECRET="your-client-secret-here"

IMPORTANT: In Google Console also enable:
- APIs & Services → OAuth consent screen
- Add your email as a test user while in development
- Scopes needed: email, profile

=============================================================
HOW TO SET UP EMAIL (MAGIC LINKS) — 3 options
=============================================================

--- OPTION A: Gmail (easiest for dev) ---
1. Go to your Google Account → Security
2. Enable 2-Step Verification
3. Then go to: https://myaccount.google.com/apppasswords
4. Create an app password for "Mail"
5. Add to .env:

EMAIL_SERVER_HOST="smtp.gmail.com"
EMAIL_SERVER_PORT="587"
EMAIL_SERVER_USER="your.gmail@gmail.com"
EMAIL_SERVER_PASSWORD="your-16-char-app-password"
EMAIL_FROM="your.gmail@gmail.com"

--- OPTION B: Resend (best for production, free tier) ---
1. Sign up at https://resend.com
2. Add and verify your domain
3. Create an API key
4. npm install nodemailer-resend  (or use their SMTP)

EMAIL_SERVER_HOST="smtp.resend.com"
EMAIL_SERVER_PORT="587"
EMAIL_SERVER_USER="resend"
EMAIL_SERVER_PASSWORD="re_your_api_key_here"
EMAIL_FROM="noreply@yourdomain.com"

--- OPTION C: Brevo / Sendinblue (free 300 emails/day) ---
1. Sign up at https://brevo.com
2. Go to SMTP & API → SMTP tab
3. Add to .env:

EMAIL_SERVER_HOST="smtp-relay.brevo.com"
EMAIL_SERVER_PORT="587"
EMAIL_SERVER_USER="your-brevo-login-email"
EMAIL_SERVER_PASSWORD="your-brevo-smtp-password"
EMAIL_FROM="noreply@yourdomain.com"
